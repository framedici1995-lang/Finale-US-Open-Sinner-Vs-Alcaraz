# 🎾 Sinner vs Alcaraz: Grand Slam US Open 2026

Un gioco di tennis in stile 8-bit realizzato in **Python** e **Pygame**. Rivivi la sfida epica tra i due campioni nel campo degli US Open!

## 🚀 Caratteristiche
- **Audio 8-bit dinamico**: Suoni generati proceduralmente via codice.
- **Sistema di punteggio reale**: Gestione di punti (15, 30, 40, AD), game e set.
- **Difficoltà variabile**: Scegli tra Facile, Normale o Leggendario.

## 🎮 Comandi
- **Frecce Su/Giù**: Muovi il tuo tennista.
- **Tasti 1, 2, 3**: Seleziona la difficoltà nel menu.

pip install pygame
python main.py
